Database
Koleksi rekaman-rekaman (data) yang terstruktur. Kumpulan rekaman ini dikelola menggunakan Sistem Manajemen Basis Data (SMDB) untuk menambah, menghapus, dan memperbarui data. Di sini data disusun menjadi Kolom (atribut) dan Baris (rekaman).
Memento Database

Sebuah aplikasi database personal untuk Android. Memento membantu merancang database dengan beragam field, metode penyajian (sort, filter), penyimpanan data secara lokal atau cloud, serta fitur lainnya.

Website Memento Database dalam bahasa Inggris. Terdapat penjelasan tentang fitur, harga, program untuk dekstop, juga tautan ke forum, wiki, dan uservoice.

Templat Memento Database

Berkas kerangka untuk aplikasi Memento Database. Berisi field-field yang sudah ditetapkan sebelumnya.

Grup
Satu kelompok koleksi, atau satu bagian dari koleksi dalam sebuah database. Grup menyederhanakan daftar kumpulan koleksi. Grup hanyalah kontainer; tidak ada kegunaan lainnya.
Koleksi
Kontainer untuk entri (rekaman, baris). Sinonim dengan tabel/worksheet dalam istilah Excel.
Field
Sinonim dengan 'Kolom' dalam istilah Excel. Field adalah elemen utama dari struktur sebuah koleksi. Satu field adalah sebuah wadah tempat satu unit data atau nilai. Tiap field punya tipe, nama, dan petunjuk. Field juga bisa memiliki parameter lanjutan atau opsi tampilan, yang tergantung tipe fieldnya. Koleksi bersinonim dengan tabel/worksheet dalam istilah Excel.
Entri
Kontainer untuk entri (rekaman data). Sinonim dengan 'Baris' dalam istilah Excel.
Tab
Batas pemisah antara satu halaman dengan halaman lain dalam satu koleksi.
Halaman
Suatu divisi sub dari satu set field pada sebuah koleksi. Halaman awal dari sebuah koleksi dalam sebuah tab disebut UTAMA. Pengguna dapat menambahkan lebih banyak halaman; masing-masing akan memiliki sebuah nama dan sebuah tab; masing-masing pula akan memiliki set field sendiri. Tiap entri dalam satu koleksi berisi seluruh field untuk semua halaman.
Action
Script-script pemograman yang menjalankan suatu tindakan secara otomatis. Untuk menghidupkannya sentuh pada Layar Entri.
Agregasi
Pada aplikasi Memento, agregasi adalah satu set operasi matematika dan tampilan yang merunut pada semua koleksi, atau masing-masing grup entri dalam satu koleksi. Operasi ini terdiri dari penjumlahan (sum), rata-rata (average), maksimum (max), dan minimum (min). Tampilannya termasuk menunjukkan entri dari koleksi yang berhubungan (dengan entri lainnya) dari koleksi yang sedang diperlihatkan.
Autofill
Pada aplikasi Memento, sebuah tindakan pengisian data dari sumber data eksternal saat pembuatan entri.
